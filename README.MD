# Recommender System

## Usage
* -m: using mass
* -h: using heat
* -H: using hybrid
* -N: using HNBI
* -E: using RENBI
* -C: using UCF
* -F: using ICF
* -S: using SVD
* -u: users extra attribute
* -i: full dataset filename
* -T: train dataset filename
* -t: test dataset filename
* -l: loop Number
* -d: dataset divide rate(double)
* -s: random seed
* -L: L
* -K: K
* -?: help information
* ./bin-app-bnr-biprecmmd -m -u ../Dropbox/ml-1m/user_p3 -i ../Dropbox/ml-1m/ratings_p2
